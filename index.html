Skills

Power BI

Power BI is a business analytics tool by Microsoft that enables users to visualize data, create interactive reports, and share insights across an organization. It integrates data from various sources to provide real-time business intelligence and decision-making capabilities.

Python

Python is a versatile, high-level programming language known for its simplicity and readability. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming, making it ideal for web development, data analysis, automation, and more.

MySQL

MySQL is an open-source relational database management system (RDBMS) that uses structured query language (SQL) to manage and manipulate data. It is widely used for building and managing databases in web applications due to its scalability and performance.

Excel

Microsoft Excel is a powerful spreadsheet application used for data organization, analysis, and visualization. It features tools for creating charts, performing calculations, and managing complex data sets.

HTML

HTML (HyperText Markup Language) is the standard language used to create and structure web pages and web applications. It uses a system of tags and attributes to define elements like headings, paragraphs, links, and images.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall visual design of web pages, enhancing their appearance and user experience.
